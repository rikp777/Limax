# Install setup

**Setup project**
- \(Optional) Rename the .env.example file to .env
- composer install
- php artisan key:generate
- php artisan storage:link
- npm install
- npm run dev
- php artisan migrate:fresh --seed
- php artisan passport:install
- composer dump-autoload

**Run project**
- php artisan serve
- npm run watch
